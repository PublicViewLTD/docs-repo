# 推送垃圾称重数据接口详细说明

- 版本: `1.0`

| 数据项   | 数据类型        | 描述       | 必须字段 | 备注 |
| -------- | --------------- | ---------- | -------- | ---- |
| `device`    | `string` | 设备ID     | `true`   |      |
| `weight` | `number` (`Decimal`) | 称重数据 | `true`   | 单位：kg |
| `trash` | `string` | 垃圾桶编号  | `true` | |
| `type` | `number` (`Int16`) | 垃圾类型 | `true`||
|`scanningTime`| `number` (`Int64`) | 数据生成时间戳 | `true`| UNIX标准时间戳 |


## 垃圾类型说明

固定枚举类型：

- 缺省类型 : `0`
- 厨余垃圾 : `1`
- 可回收物 : `2`
- 有害垃圾 : `3`
- 其他垃圾 : `4`

> 尽量请勿传递缺省类型
